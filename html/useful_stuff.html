Dies ist eine Liste ausgewählter nützlicher Freeware, welche sowohl schnell und einfach zu downloaden als auch zu installieren ist.
  • Notepad++
    Ein nützliches Textverarbeitungsprogramm. Wird hauptsächlich zum Programmieren verwendet, kann aber auch als simpler "Editor/Notepad"-Ersatz dienen.
  • Opera Web-Browser
    Der Opera Web-Browser bietet mehr Kompatibilität zu jeglichen File-Formaten als der Internet Explorer und ist zudem um einiges benutzerfreundlicher als der allseits bekannte Firefox Web-Browser.
  • Xampp Windows-Server
    Der Xampp Server bietet ein tolles Rundum-Paket jeglicher Art von Server. Mit dabei sind:
    Apache Server (für Websites)
    MySQL Server (für die Datenbankverwaltung)
    PHP5 Server (für PHP-Unterstützunug auf den Websites)
    FileZilla FTP Server (für FTP-Hostings)
